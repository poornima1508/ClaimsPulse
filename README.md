# ClaimsPulse

A local decision support tool that analyzes healthcare insurance claims data and helps leadership understand cost trends.

## What This Does

ClaimsPulse takes synthetic healthcare claims data and turns it into a simple dashboard that shows:

- Total claims and costs at a glance
- Which claim categories are driving the most cost
- Which providers have the highest billing
- Months where costs spiked unusually high
- A short AI-generated executive summary of the findings

## Why I Built This

Insurance companies deal with large volumes of claims data every day. Leadership often struggles to understand what is driving cost increases without spending hours going through spreadsheets. This tool aims to make that process faster and clearer.

## Tools Used

- Python
- Pandas and NumPy for data processing
- Streamlit for the dashboard
- Synthea for generating synthetic patient data
- Qwen (via Ollama) for the executive summary

## How to Run It

1. Clone the repository
2. Create a virtual environment and activate it
3. Install dependencies: pip install pandas numpy streamlit
4. Install Ollama and pull the Qwen model: ollama pull qwen2:1.5b
5. Run the dashboard: streamlit run app.py

## Data

This project uses synthetic data generated by Synthea. No real patient data was used.
